# Notes Manager


## Περιγραφή
Python web app με Flask που επιτρέπει δημιουργία, προβολή και διαγραφή σημειώσεων.
Το πρόγραμμα αποθηκεύει τις σημειώσεις στο αρχείο notes.txt.


## Οδηγίες Χρήσης
1. Βεβαιώσου ότι έχεις εγκατεστημένη Python 3.
2. Εγκατέστησε Flask:
	Άνοιξε το CMD και γράψε pip install flask
3. Άνοιξε το πρόγραμμα:cmd
4. Γράψε cd και την τοποθεσία του αρχείου π.χ cd C:\Users\Tony\Notes Manager
5. Γράψε την εντολή: python notes_manager.py
6. Άνοιξε το browser και πήγαινε στην σελίδα http://127.0.0.1:5000/


## Πρόσθεσε, δες ή διαχειρίσου τις σημειώσεις.
